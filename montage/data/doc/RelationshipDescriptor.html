criteriaExpressions :Object.<string, external:FrbExpression>

The criteria to set on the selector used to fetch the objects returned by this relationship. These criteria are expressed in terms of the names of the criteria to set on the selector, and for each of these names an FRB expression defining how the corresponding criteria value will be obtained. Each of these expressions will be evaluated within the scope of objects having the relationship to get the criteria value.

The object provided by this property should not be modified by any object other than RelationshipDescriptor.

valueExpressions :Object.<string, external:FrbExpression>

The values to be set by the relationship on objects having the relationship. These values are expressed in terms of the names of the properties whose values are to be set, and for each of these names an FRB expression defining how the corresponding property value will be obtained. Each of these expressions will be evaluated within the scope of the objects returned by the relationship to get the property value.

The object provided by this property should not be modified by any object other than RelationshipDescriptor.
