Class: DataMapping

DataMapping

new DataMapping()

Maps raw data to data objects of a specific type.

Currently services have to subclass this and override its mapData() method to define their mapping. In the future mappings will be defined declaratively through mapping descriptors read from blueprint files.

Methods

mapRawData(rawObject)

Convert raw data to data objects of an appropriate type.

Subclasses should override this method to create an object of the right type with values taken from the passed in raw object. The specialize() method of the prototype of the desired object type can be used for this, as in the following:

mapRawData: {
    value: function (rawObject) {
        return MyType.prototype.specialize({
            firstName: {
                value: rawObject.GIVEN_NAME
            }
            lastName: {
                value: rawObject.FAMILY_NAME
            }
        });
    }
}

The default implementation of this method returns the raw object unmodified.

Parameters:
Name Type Description
rawObject Object

An object whose properties hold the raw data. This object may be modified.
