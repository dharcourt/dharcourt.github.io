Methods

addRawData(stream, objects)

To be called by getRawData() when raw data is received.

This method maps the raw data to data objects using the specified service's mapping, registers those objects with the service so it can manage them, and then adds those objects to the specified stream.

Parameters:
Name Type Description
stream DataStream

The stream to which the data objects corresponding to the raw data should be added.
objects Array

An array of objects containing the raw data. This array and the objects it contains may be modified.

getData(type, selector, stream)

Get data from the service. This is the main method used by clients of this service.

This method gets raw data from a server or other source using the getRawData() method, maps that data to data objects that fit into an application's data model using the service's mapping, registers those objects so the service can manage them, and then return those objects in the specified stream or in a new stream created for this purpose.

Parameters:
Name Type Description
type ObjectDescriptor

The type of the data requested.
selector DataSelector

Criteria that the returned data objects must satisfy.
stream DataStream

The stream to which the provided data should be added. If not stream is provided a stream will be created and returned by this method.

getRawData(stream)

Get the raw data used by this service.

This class' implementation simply calls rawDataDone() immediately

rawDataDone(stream)

Indicate that all the raw data meant for the specified stream has been added to that stream.

Subclasses must call this method once with appropriate values for each getRawData() call they receive.
